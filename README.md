# Escaper
javascript 实现的射击游戏

按键

键盘|行为
:-:|:-:
w / up arrow | move up
s / down arrow | move down
a / left arrow | move left
d / right arrow | move right
q / e | change weapon
r | reload
blankspace / mouse click | fire 
enter | pasue
n | new a monster(test mode)
m | new a reward(test mode)
